## 🛠️ Libraries Covered

### 1. NumPy
**The Foundation of Numerical Computing.** NumPy is essential for handling N-dimensional arrays and matrices. It provides a massive collection of high-level mathematical functions to operate on these arrays efficiently.
* **Focus:** Vectorization, Linear Algebra, and Fourier Transforms.
![Numpy](https://github.com/dhrumilrana25/Python-Libraries-Tutorial/assets/139121688/1046584e-54b5-4448-83bf-b731c069752d)

### 2. Pandas
**High-Performance Data Structures.** Pandas is the go-to library for data "wrangling." It introduces the **DataFrame**, making it simple to manipulate tabular data, handle missing values, and perform time-series analysis.
* **Focus:** Data Cleaning, Merging/Joining, and Exploratory Data Analysis (EDA).
![Pandas_logo svg](https://github.com/dhrumilrana25/Python-Libraries-Tutorial/assets/139121688/1bbe69e6-43a8-4608-99ae-97173b598901)

### 3. Matplotlib & Seaborn
**Turning Data into Insights.** * **Matplotlib:** The foundation for static, animated, and interactive visualizations.
* **Seaborn:** Built on top of Matplotlib, it provides a high-level interface for drawing attractive and informative statistical graphics.
![matplot_title_logo](https://github.com/dhrumilrana25/Python-Libraries-Tutorial/assets/139121688/7f321118-96e8-4c4c-bc97-9ef7d6c9477b)
![seaborn](https://github.com/dhrumilrana25/Python-Libraries-Tutorial/assets/139121688/84ed1374-1afe-4375-b5c9-e9e45bd07204)

### 4. Keras & TensorFlow
**Deep Learning at Scale.** An open-source end-to-end platform for machine learning. This tutorial covers building, training, and deploying neural networks using the Keras high-level API.
* **Focus:** CNNs, RNNs, and Model Optimization.
![keras](https://github.com/dhrumilrana25/Python-Libraries-Tutorial/assets/139121688/f0115c96-6e85-4e9b-9836-98c9d5775fc6)
![tensor flow](https://github.com/dhrumilrana25/Python-Libraries-Tutorial/assets/139121688/b3d3926f-2ac2-4f44-a80f-cc8f1e8799d6)

## 🚀 How to Use
Each library is contained within its own dedicated directory.
1. **Clone the repo:** `git clone https://github.com/dhrumilrana25/Python-Libraries-Tutorial.git`
2. **Navigate** to the specific library folder (e.g., `/Pandas`).
3. **Open the Jupyter Notebooks** or `.py` files to follow along with the code and explanations.
